# NetEase_Cloud_Music_Client
It is designed to be a simplified cross-platform PWA implementation of NetEase Cloud Music.

## Project Anatomy
* PWA
* Vue.js
* Vue-Router
* Vuex
* Vuetify
* SASS/SCSS
    * node-sass
    * node-loader
* ESLint
    * standard preset
* Mocha & Chai  


## Project setup
```
yarn install
```
